# 黄焖鸡的介绍
黄焖鸡（Chicken braised with brown sauce），又称香鸡煲、浓汁鸡煲，是山东省济南市的传统鲁菜，属于家常菜品。该菜品以鸡腿肉为主料，搭配青椒、土豆、香菇等食材，通过炒、焖等工艺制成，具有香辣鲜的口味特征。其起源相传与明代朱元璋食用三黄鸡故事相关；亦有称其在清朝嘉庆年间曾成为宫廷菜。1927年济南名店吉玲园推出“百草黄焖鸡”获时任山东省主席韩复榘赞誉并赐名。该菜品以“一品汤、二品肉、三品米”为特色，焖制过程注重火候把控，鸡汁鲜香浓郁，肉质滑嫩弹牙。2011年，黄焖鸡被发展为标准化快餐模式，推动了其在全国的普及。鸡肉富含优质蛋白质及人体必需氨基酸，香菇、土豆等配料兼具营养与提味功能。根据现代营养学分析，黄焖鸡热量较高，但鸡肉提供优质蛋白，减肥期间亦可适量食用。作为国民小吃代表，黄焖鸡米饭常配以米饭，形成独特的风味体系。黄焖鸡米饭与沙县小吃、兰州拉面并称为中国三大连锁

# 黄焖鸡的做法

黄焖鸡是一道十分下饭的美食，食材平平无奇又十分容易烹制，一学就会。

预估烹饪难度：★★★

## 必备原料和工具

- 鸡腿
- 香菇（干香菇最佳）
- 青椒
- 生姜片
- 干辣椒
- 盐
- 料酒
- 白胡椒粉
- 白糖
- 酱油
- 味精
- 土豆（可选）

## 计算

- 鸡腿 = 两只
- 香菇（干香菇最佳）= 5 朵
- 青椒 = 两个
- 生姜片 = 两片
- 干辣椒 = 5，6 个
- 盐 = 10g
- 料酒 = 10ml
- 白胡椒粉 = 5g
- 白糖 = 5g
- 酱油 = 5ml
- 土豆 = 一个（可选，可使汤汁更粘稠）

## 操作

### 备菜阶段

- 鸡腿洗净，剁成**4cm**大小的块
- 生姜切片、干辣椒切成**小圈**
- 香菇切片，青椒切成细长的**马蹄状**，若为干香菇，洗净灰尘后泡一晚上并留香菇水备用
- 若有土豆，切为与鸡肉大小类似的**滚刀块**

### 炒制阶段

- 炒糖色：锅里倒入底油，冷油时放入白糖（**有一定难度，新手可跳至鸡肉炒制并使用老抽替代**）
- 小火慢慢加热，待油温逐渐升高，白糖开始融化并变成较深的棕色（期间要不断搅拌，防止糊锅）
- 迅速倒入鸡块，转大火，快速翻炒！烹入料酒，继续翻炒片刻
- 将生姜片和干辣椒倒入
- 放入酱油，炒匀
- 倒入香菇水或清水，以能淹住鸡肉为准
- 倒入香菇片，白胡椒粉，盐，土豆
- 翻炒均匀后，盖上锅盖焖煮，转中小火**15——20分钟**，有条件可以转至砂锅
- 待鸡肉软烂，汤汁浓稠后（汤汁不要收的太干），最后放入青椒
- 放入味精，兜炒均匀后，关火！青椒基本断生即可，不要炒时间久了
